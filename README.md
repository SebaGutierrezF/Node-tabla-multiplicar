# Notas
Tabla de multiplicar Udemy node
```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Es la base de la tabla de multiplicar      [número] [requerido]
  -h, --hasta    Es el limite de la tabla de multiplicar    [número] [requerido]
  -l, --listar   muestra la trabla en consola        [booleano] [defecto: false]
  -c, --colors   Cambia el color del listado         [booleano] [defecto: false]
  ```
